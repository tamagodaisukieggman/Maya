■概要
Cygames プロジェクトに必要なMayaツールをMayaメニューに追加します。
・キャラユーティリティ
・法線ツール

■ツールインストール
zipを解凍し、フォルダを任意のフォルダに配置してください。
（配置フォルダは任意の場所ですが、パスに日本語を含めないでください）

■maya起動手順
maya2022.bat　からmayaを起動
Mayaのメニューに「Cygames」メニューが追加されます。

maya2022.batを実行した際、Microsoft Defenderの「WindowsによってPCが保護されました」といったポップアップが出てMayaが起動できない場合の対処法：
・maya2022.batを右クリック、プロパティ > 「全般」タブの一番下の「セキュリティの」項目を見つけてください。「
・このファイルは他のコンピュータから取得したものです。　このコンピュータを保護するため、このファイルへのアクセスはブロックされる可能性があります」という文言の横の「許可する」というチェックボックスにチェックを入れOK（適用）してから再度maya2022.batの実行を試してみてください。

■mayaをCドライブにインストールしていない場合
maya2022.batをテキストエディタなどで開き
set MAYA_INSTALL_PATH=C:\Program Files\Autodesk
の記載部分をMayaのインストールパスに変更してください。

■不具合や不都合がある場合はご相談ください。
