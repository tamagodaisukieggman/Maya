#＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝
使い方
#＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝
■"Symmetry" ボタンがOnの状態になると、対象の位置にある
　コントローラーも同時に選択されるようになります。

■"<=all" ボタンは、上唇用と下唇用の2つがあり
　それぞれのコントローラーを一括選択できます。

■基本的にはNamespaceを選択してボタンを押すことで
　コントローラーが選択されます。

■NameSpaceのアイテムをダブルクリックすると
　コントローラーがViewにフィットするように
　フォーカスが移動します。

■Shiftを押しながらボタンをクリックすると
　追加選択になります。

■ボタンを範囲選択で囲んで複数のコントローラーを
　同時に選択することもできます
